# 📡 Webhook Listener Pro

Лёгкий и быстрый сервис для приёма, логирования и проверки входящих вебхуков.

## 🚀 Возможности

- 🔥 Приём любых POST /webhook событий  
- 🧾 Автоматическое логирование тела запроса  
- 🛡️ Возможность проверки сигнатур  
- 📤 JSON-ответ { "status": "received" }  
- 🧩 Интеграция с GitHub, Telegram, Stripe, Notion и др.
